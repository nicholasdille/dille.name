## Internals

### Namespaces

* Used for resource isolation
* Isolation of resource usage to limit visibility
* Types are PID, network, mount

### c(ontrol)groups

* Used to limit resource usage for proceses
* Limits and measures access to...
* ...CPU, memory, network, IO
