## Artifacts

### Simple artifact

1. Create repository called `artifact` from `java-hello-world`
1. Add repository to drone and add secrets `WEBDAV_USER` and `WEBDAV_PASS`
1. Add `1/.drone.yml`
1. Check build
1. Check WebDAV

### Structured artifacts

1. Update build instructions with `2/.drone.yml`
1. Check build
1. Check WebDAV
